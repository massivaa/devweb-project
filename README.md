1. DESCRIPTION DU SITE
Le site SantéClick est une plateforme web de santé permettant la gestion et l'interaction avec les services de santé numériques qu'offre le site.
Parmis les fonctionnalités principales :
  * Système d’achat de médicaments en ligne(e-commerce).
  * Création et gestion de compte utilisateur.
  * Gestion du profil médical (poids, taille, groupe sanguin, etc...).
  * Consultation et gestion des rendez-vous médicaux.
  * Accès aux ordonnances médicales.
  * Consultation du dossier médical.
  * Recherche et consultation de médicaments.
  * Section FAQ et pages d’informations: CGU, confidentialité, mentions légales (toutes dans footer).
  * Contact en bas de la page (footer) interactif (sur  instagram, ou facebook, ou twitter, ou tiktok ou sur e-mail!).
hébergement du site: 
* Frontend : Vercel
* Backend & base de données : AlwaysData


2. INSTRUCTIONS D’UTILISATION
Après l'accée au site via le lien de déploiement (https://santeclick.vercel.app).
 * Naviguer via la barre de menu
    - vers les differentes pages: acceuil, médicaments, docteurs, profil.
    - ou bien changer de mode: sombre/claire.     
 * Créer un compte utilisateur ou se connecter (ou bien utiliser le compte test qui se trouve dans la page connexion).
 * Compléter son profil médical dans la section Profil (vous pouvez ajouter ou supprimer des rendez-vous, modifier les informations du patient...).
 * Vous pouvez acheter des médicaments sans ordonnance, ou avec une ordonnance en clickant sur l'une des ordonnances dans la section Nouvelle ordonnance dans la page Profile.
